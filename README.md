A Python project to build a phonetic dictionary for Telugu words extracted from Wiktionary. The dictionary stores words with their pronunciations in IPA (International Phonetic Alphabet) format and supports export for G2P training, and pronunciation lookup.

What is G2P? G2P stands for Grapheme to Phoneme conversion (the computational process of converting written text (graphemes) into its corresponding phonetic or phonemic transcription) which is important for Text to speech recognition and understanding the language deeply.

✅ Features:
- Download Telugu words and pronunciations from Wiktionary
- Extract IPA transcriptions
- Capture definitions and word types
- Handle network errors gracefully

✅ How Data was cleaned:
- Normalized Telugu Unicode text
- Validated IPA pronunciations
- Removed duplicates
- Comprehensive error reporting

Contributions are welcome! Areas for improvement:
- Better IPA extraction algorithms
- Integration with other phonetic sources
- Integration with larger Telugu corpuses and speech recognition features
